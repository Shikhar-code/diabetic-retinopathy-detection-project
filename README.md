# diabetic-retinopathy-detection-project
Deep learning project for detecting diabetic retinopathy from retinal images using CNN and EfficientNet transfer learning. Includes training pipeline, evaluation metrics, and confusion matrix visualization.

# Diabetic Retinopathy Detection using Deep Learning

This project uses convolutional neural networks and transfer learning to detect diabetic retinopathy from retinal images.

## Dataset
APTOS 2019 Blindness Detection Dataset  
https://www.kaggle.com/datasets/sovitrath/diabetic-retinopathy-224x224-2019-data

## Models Implemented

1. Custom CNN (baseline)
2. Transfer Learning using EfficientNetB0

## Results

| Model | Accuracy |
|------|------|
CNN Baseline | 74.6% |
EfficientNetB0 | 78.4% |

## Evaluation

- Confusion Matrix
- Classification Report
- Precision / Recall / F1 Score
